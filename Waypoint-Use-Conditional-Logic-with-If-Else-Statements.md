# Use Conditional Logic with If-Else Statements
We can use if statements in JavaScript to only execute code if a certain condition is met.

if statements require some sort of `boolean` condition evaluate.

```js
Example:

 if (1 == 2) {
  return(true);
}
else {
  return(false);
}
```
